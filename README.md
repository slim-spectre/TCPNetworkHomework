Тлумачення станів TCP з'єднань:
LISTEN: Порт відкритий і очікує на вхідні підключення (режим сервера).
ESTABLISHED: Активне з'єднання, встановлене між двома вузлами; відбувається обмін даними.
TIME_WAIT: З'єднання закрите, але ОС тримає порт зарезервованим для обробки можливих запізнілих пакетів.
CLOSE_WAIT: Віддалений вузол ініціював закриття з'єднання, ОС очікує на команду закриття від локальної програми.
