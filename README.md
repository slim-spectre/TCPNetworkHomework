### Тлумачення станів TCP з'єднань

> 

1.  **LISTEN** — Порт відкритий програмою і очікує на вхідний запит (режим сервера).
2.  **ESTABLISHED** — Канал зв'язку встановлено, відбувається активний обмін даними.
3.  **TIME_WAIT** — З'єднання завершене, але ОС чекає певний час, щоб прийняти пакети, які могли затриматися в мережі.
4.  **CLOSE_WAIT** — Сервер ініціював закриття, локальна машина готується до завершення сесії.
